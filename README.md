# SDV Security and Privacy Elicitation

This repository contains the questionnaire used in the academic study on
**Security and Privacy of Software-Defined Vehicles (SDVs)** (https://arxiv.org/abs/2411.10612).

## Contents
  Full list of elicitation questions used to collect expert opinions from industry.

## Purpose
The survey investigates:
- SDV architectural characteristics
- Security threats and attack surfaces
- Mitigation strategies
- OTA security and non-functional properties

## Usage
The questionnaire is made publicly available to support:
- transparency
- reproducibility
- comparison with future studies

## Citation
If you use this survey, please cite the associated paper.

## Anonymity
Only survey questions are published. No personal data or responses are shared.
