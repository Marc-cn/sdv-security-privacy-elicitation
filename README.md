# SDV Security and Privacy Elicitation

This repository contains the questionnaire used in the academic study on
**Security and Privacy of Software-Defined Vehicles (SDVs)**:

De Vincenzi, M., Pes√©, M. D., Bodei, C., Matteucci, I., Brooks, R. R., Hasan, M., Saracino, A., Hamad, M., and Steinhorst, S. (2024). Contextualizing Security and Privacy of Software-Defined Vehicles: State of the Art and Industry Perspectives. arXiv:2411.10612. https://arxiv.org/abs/2411.10612

## Contents
  Full list of elicitation questions used to collect expert opinions from industry.

## Purpose
The elicitation investigates:
- SDV architectural characteristics
- Security threats and attack surfaces
- Mitigation strategies
- OTA security and privacy

## Usage
The questionnaire is made publicly available to support:
- transparency
- reproducibility
- comparison with future studies

## Citation
If you use this survey, please cite the associated paper (https://arxiv.org/abs/2411.10612).

## Anonymity
Only elicitation questions are published. No personal data or responses are shared.
